# homebrew-tap

Homebrew tap for [flacoAi](https://github.com/Roura-io/flaco) — a local AI CLI powered by Ollama.

## Installation

```bash
brew tap Roura-io/tap
brew install flacoai
```

## Requirements

flacoAi requires [Ollama](https://ollama.com) to be running locally:

```bash
brew install --cask ollama
```
